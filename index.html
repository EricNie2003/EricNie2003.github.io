Minecraft-Server

前置知识

  1. Minecraft 常见的版本有 Java 版与基岩版两种，我们联机使用 Java 版本（一直玩的 Java 版，懒得研究基岩版了。而且 Java 版可以玩盗版）
  2. Java 版有若干版本号，我们联机用 1.19.3
  3. Minecraft 可以通过安装多种 mod 来丰富游戏体验，不过没有 mod 的纯净版本也可以玩
  4. 更多知识请参阅我的世界 Minecraft Java 版 下载指南 | 文件结构说明 | 推荐启动器

系统需求

下图来自我的世界 Minecraft Java 版 下载指南 | 文件结构说明 | 推荐启动器。其实只要不是非常古老的电脑应该都行。

准备工作

  1. 下载启动器
    下面以 HMCL 启动器为例，也可以选择自己喜欢的启动器
    在这个链接下载，选择第一个 exe 文件进行下载，放到一个单独的文件夹中
  2. 打开启动器，如果没有 Java 的话，据说会提醒要不要下载，选择下载即可
  3. 新建账户
    • 登录方式：离线模式（其实就是盗版 x）
    • 用户名：随便填，支持中文（但最好别带中文）
  4. 下载并安装游戏本体
    1. 点击左侧的游戏列表
    2. 选择上方“全局游戏设置” - 第二项“运行路径”，选择“各版本独立”
    3. 点击左上角返回后，选择上方“安装新游戏版本”
    4. 找到自己需要的版本号
    5. 名称随便起一个，自己能认出来就行。底下的全都不要管
      如果下载卡住，在选择版本的页面右上角更换一个下载源
  5. 点击左上角返回主菜单，点击右下角启动游戏。如果“启动游戏”下的游戏名称与你刚刚设置的不同，点击红框框出的箭头，选择正确的游戏版本。
  6. 等待游戏启动。可能要下载一些东西，需耐心等待一段时间
  7. （可选）自己安装想要的 mod

    安装可以使用 mod 的版本

    先学会如何安装纯净版
  8. 仿照安装纯净版的过程，进行到下图所示步骤
  9. 若无特殊需求，Fabric版本选择不安装，Forge版本可以无脑选最新，LiteLoader版本选择不安装，OptiFine版本也可以无脑选最新

如何联机

  1. 选择“多人模式”
  2. 选择“添加服务器”
  3. “服务器名称”随意，服务器地址微信私聊找我进群看群公告

服务器说明

大厅

大厅供大家选择不同的服务器进行游玩，同时也有一定量的（抽象）建筑可供参观
选择服务器的方式是：右键下图所示位置的告示牌，打开选择界面后，挑选自己要去的服务器；退服务器重进后会回到大厅

生存服

生存服中安装了领地和商店插件，可以通过交易得到一定的钱之后，购买土地

基础指令

使用/tpa <玩家>请求传送到指定玩家身边
使用/t <玩家> <信息>与指定玩家私聊

经济相关

玩家间交易

使用/pay <玩家> <数额>可以给指定玩家指定数额的钱

商店插件

见插件原帖 How to create a shop 和 How to buy/sell Items 两部分

服务器提供了一个官方市场从玩家处不限量收取货物，作为玩家的主要经济来源

领地相关

使用/res获得帮助，或者看这个链接

zerotier

干啥用的

内网穿透要到期了（就是说之前的服务器那个地址就用不了了），并且假期余额不足一个月所以我懒得续费了。而且它老是间歇性出bug把所有人都卡下去实在是太屑了。

这玩意有一定概率能够代替内网穿透（至少我和zcc测试的情况是可以，然而每个人家里网络情况不一样所以不能保证100%都行）。如果成功的话，一方面会比较稳，一方面延迟理论上会更低一点，同时带宽也许会更大一些（我猜测大家家里的带宽应该都比内网服务器的100M大吧）

如何下载

链接: https://pan.baidu.com/s/1g2ch8m9yCqtPfSbhzVkcmA?pwd=m8py 提取码: m8py
解压密码同文件名（只是因为担心度娘抽风所以弄了个密码）

如何使用

  1. 安装
  2. 在右下角找到它的小图标
  3. 右键-open control panel
  4. 在Join Network按钮左边的框里输入565799d8f611bc1f，点击Join Network按钮
  5. 操作完跟我说一声，我这边通过一下加入的请求
  6. 以后就可以用10.147.18.227:25565进服了

steam++

什么是Steam++

Steam++有如下功能（从官网复制粘贴的）

加速访问

该功能能够使 Steam 社区、 Github 、谷歌验证码等国内难以访问的网页正常访问。功能类似 steamcommunit302 ，使用 Titanium-Web-Proxy 开源项目进行本地反代。

内置插件

该功能内置了大量的游戏相关油猴脚本： Steam 显示史低价格、一键出售 Steam 库存物品、跨区汇率换算、批量激活 CdKey 。脚本可以直接植入 Steam 客户端，体验比原先的脚本更为优秀。用户也可以制作自己的脚本导入软件进行使用。

库存管理

该功能能够让您直接管理到 Steam 游戏库存，不仅能够下载老版本的 Steam 游戏，也能够对 Steam 游戏成就进行直接管理，支持解锁成就以及反解锁成就。

令牌管理

该功能能够让您的手机令牌统一保存在电脑中，并且支持云备份，支持的令牌有 Steam 令牌、暴雪令牌、 Uplay 令牌、 Epic 令牌、 R 星令牌等常规 6 位、 8 位令牌。

下载地址

官网链接：https://steampp.net/
选择Windows版下载即可（我相信大家都用的是Windows）
